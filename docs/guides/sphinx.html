Introduction

The Sphinx documentation is written using the reStructuredText .rst language (though it can also be written with the Markdown .md language).

Getting started

In the root of the project:

../pygen/
    docs/...
    doc-source/
    pygen/...
    .gitignore
    cli.py
    LICENSE
    README.md
    requirements.txt
    setup.py
    TODO.md

and run the following command:

1$ sphinx-build doc-source/ docs/

or

1$ sphinx-autobuild doc-source/ docs/

See Next section about sphinx-autobuild.

Important

docs/ should be built with the previous command, otherwise Github Pages won’t be able to read the files properly. A .nojekyll file should also be added in the docs/ folder if it is not already present.

Sphinx autobuild

Leave the docs/ folder, then run the documentation on a local server which listens to any changes to the doc and updates them live.

1$ cd ..
2
3$ sphinx-autobuild doc-source/ docs/

Sphinx themes

Switching themes

To change the theme, simply change the value html_theme = "sphinx_rtd_theme" in conf.py to any other theme. You must also include the theme in your setup.py/requirements.txt files.
