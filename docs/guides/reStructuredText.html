Making a Glossary

 1.. glossary::
 2
 3    environment
 4        A structure where information about all documents under the root is
 5        saved, and used for cross-referencing.  The environment is pickled
 6        after the parsing stage, so that successive runs only need to read
 7        and parse new and changed documents.
 8
 9    source directory
10        The directory which, including its subdirectories, contains all
11        source files for one Sphinx project.

Example:

environment

A structure where information about all documents under the root is saved, and used for cross-referencing. The environment is pickled after the parsing stage, so that successive runs only need to read and parse new and changed documents.

source directory

The directory which, including its subdirectories, contains all source files for one Sphinx project.
